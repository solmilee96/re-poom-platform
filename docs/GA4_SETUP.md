# GA4 연동 방법 (Google Analytics 4)

## 1단계: GA4 측정 ID 발급

1. **Google Analytics 접속**  
   https://analytics.google.com/ 에 로그인합니다.

2. **계정/속성 만들기**
   - 왼쪽 아래 **관리(톱니바퀴)** 클릭
   - **만들기** → **계정** 이름 입력(예: 아이 여행) → **속성** 이름 입력(예: re-poom-platform) → 산업/규모 선택 후 **만들기**

3. **데이터 스트림 추가**
   - 플랫폼에서 **웹** 선택
   - 웹 사이트 URL: `https://solmilee96.github.io/re-poom-platform/`
   - 스트림 이름: 예) re-poom-platform
   - **스트림 만들기** 클릭

4. **측정 ID 복사**
   - 만들어진 스트림을 클릭하면 상단에 **측정 ID**가 보입니다.  
     형식: `G-XXXXXXXXXX`  
   - 이 값을 복사합니다.

---

## 2단계: 코드에 측정 ID 넣기

1. **docs/index.html** 파일을 엽니다.

2. **G-XXXXXXXXXX** 를 검색해서 **모두** 본인이 복사한 측정 ID(예: G-AB12CD34EF)로 바꿉니다.  
   - 총 2곳입니다(스크립트 src의 id, config의 id).

3. 저장 후 GitHub에 푸시하면 GA4 수집이 시작됩니다.

---

## 3단계: GA4에서 보는 방법 (이탈/행동)

- **보고서** → **참여** → **페이지 및 화면**  
  - `screen_name` 기준으로 단계별 조회 수를 볼 수 있습니다.

- **탐색** → **퍼널 탐색** (또는 **자유 형식**)  
  - 차원에 **페이지 및 화면** 또는 **이벤트** → `screen_view`  
  - 단계별 유저 수·이탈 구간을 확인할 수 있습니다.

- 수집은 배포 후 **최대 24~48시간** 지나면 보고서에 반영될 수 있습니다. 실시간은 **보고서** → **실시간**에서 확인 가능합니다.
