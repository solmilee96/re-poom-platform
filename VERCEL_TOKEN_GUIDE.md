# 🔑 Vercel 토큰 생성 가이드

## 단계별 안내

### 1단계: 토큰 생성 페이지 접속
👉 https://vercel.com/account/tokens

### 2단계: 토큰 생성
1. "Create Token" 버튼 클릭
2. 토큰 이름 입력: `re-poom-platform` (또는 원하는 이름)
3. Scope (권한) 설정:
   - ✅ Full Account (기본값으로 체크되어 있음)
4. "Create Token" 클릭

### 3단계: 토큰 복사
⚠️ **중요**: 토큰은 **한 번만** 표시됩니다!
- 생성된 토큰을 복사하세요
- 안전한 곳에 보관하거나 바로 사용하세요

### 4단계: 토큰 전달
토큰을 생성하셨다면, 토큰을 알려주시면 제가 바로 프로젝트를 생성하겠습니다!

---

## 토큰 예시 형식
토큰은 보통 다음과 같은 형식입니다:
- `vercel_xxx...` 또는
- 다른 형식의 긴 문자열

토큰을 생성하셨다면 알려주세요! 🚀
