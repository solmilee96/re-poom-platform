# 베이비마켓 🍼

우리 동네 육아템 나눔터 - 중고거래 플랫폼

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

## 주요 기능

- 🛍️ 중고거래: 아기 용품 중고 거래
- 📅 대여: 육아템 대여 서비스
- 👥 커뮤니티: 육아맘 커뮤니티
- 👤 마이페이지: 내 거래 관리

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Lucide React (아이콘)
